# Exahia

Open-source building blocks for sovereign enterprise AI in Europe.

## What we publish

- [`pii-detector-fr`](https://github.com/Exahia/pii-detector-fr)  
  French PII detection/anonymization library + CLI.
- [`llm-benchmark-fr`](https://github.com/Exahia/llm-benchmark-fr)  
  Reproducible benchmark runner for French business tasks.
- [`shadow-ai-audit`](https://github.com/Exahia/shadow-ai-audit)  
  Shadow AI audit toolkit: checklists + scoring CLI.
- [`exahia`](https://github.com/Exahia/exahia)  
  Product and architecture overview.

## Why it matters

Teams in regulated sectors need AI productivity without losing control over sensitive data.
Our repos focus on practical security/compliance tooling that can be run and audited locally.

## How to contribute

1. Pick a repository and open an issue.
2. Look for labels like `good first issue`.
3. Submit a PR with reproducible validation steps.

To help the project:
- Star the repos you use
- Open bug reports with reproducible examples
- Share feedback from real deployments
