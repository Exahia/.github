# Contributing to Exahia Open Source

This repository defines community defaults for the `Exahia` organization.

## Where to contribute

- `pii-detector-fr`: French PII detection/anonymization tooling
- `llm-benchmark-fr`: reproducible benchmark runner
- `shadow-ai-audit`: audit checklists and scoring tools

## Contribution workflow

1. Open an issue first for non-trivial changes.
2. Fork and create a focused branch.
3. Add reproducible validation steps in your PR.
4. Keep changes small and reviewable.

## Quality expectations

- No secrets or personal data committed to the repo
- Reproducible commands for tests/benchmarks
- Clear documentation for behavior changes

## Code of conduct

By participating, you agree to follow [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
